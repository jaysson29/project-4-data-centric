    • How To Add A Game ?

      Step By Step Guide :

      • 1. Firstly Select The Manage Games Option From The Admin Drop Down Menu On The Navigation Bar.
      • 2. Secondly Click The New Game Button at the bottom of the page.
      • 3. This Will Take You To A Form, Which Will Allow You To Enter Details About A Game You Want To Add.
      • 4. When Selecting The Platform It Will Have A Drop Down Menu With Multiple Platforms, Select Your Desired Platform.
      • 5. When Providing The Link To The Trailer You Will Have To Do The Following:
      • 6. Go To Youtube And Find A Trailer For The Game You Want Then Refer To The URL / Website Address.
      • 7. Copy The Code From The Url After the = Character And Paste It In The Game Trailer Box
      • 8. Finally Once All Information Has Been Correclty Entered Press "Add Game" Which Will Add It To The DataBase And Will Be Avaible On The Games Section For Every One To See.

    • How To Edit A Game ?

      Step By Step Guide :

      • 1. Select The Manage Games Option From The Admin Drop Down Menu On The Navigation Bar.
      • 2. Click The Game You Want To Edit.
      • 3. Press The "Edit" Buttom To Edit Game
